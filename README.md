# TextFightingGame
这是一个基于Java的文字版回合制格斗游戏，主要在控制台进行操作，包括登录、注册业务，以及玩家属性点分配系统。

玩家初始血量80HP，初始攻击力12ATK，初始防御力6DEF，初始蓝量80，初始最大血量和初始最大蓝量即为初始血量和初始蓝量，创建角色时共有10点属性点可供分配，可分配给血量、攻击力、防御力，每点属性点+5HP，+1ATK，+1DEF，若输入负数则默认分配0点到当前属性，若超出剩余点数则默认把剩余属性点全部分配到当前属性，玩家有三个技能，分别为普通攻击（无消耗，造成基础伤害）、强力一击（消耗25蓝量，造成180%攻击力的伤害）、生命回复（消耗20蓝量，随机恢复12-25点HP）。每回合结束恢复10蓝量，每打败一个敌人恢复40%最大生命值，50%最大蓝量，每胜利三场最大生命值+20，攻击力+3，防御力+2，最大蓝量+15。

敌人有四个，分别是初级战士（血量95，攻击力14，防御力7，技能猛击（130%伤害））、敏捷刺客（血量80，攻击力13，防御力6，技能快速攻击（2次80%伤害））、重装坦克（血量130，攻击力11，防御力10，技能防御姿态（下回合所受伤害减半））、神秘法师（血量70，攻击力17，防御力5，技能火球术（170%伤害）），每场敌人属性会有所增强，血量+5，攻击力+2，防御力+1，从第二场开始，敌人普通攻击和技能攻击每回合都有50%的概率释放。

src文件里的App.java是整个游戏的入口
